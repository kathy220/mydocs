# 设置settings.json
python3.8对应的环境地址是：D:/Anaconda/envs/py38/python.exe
手动配置python解释器的路径到settings.json里
    左下角齿轮-设置
    搜索python path
    设置Python: Default Interpreter Path

这样就不需要修改powershell安全策略了，更安全（见上一条）
